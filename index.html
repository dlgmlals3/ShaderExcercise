그래프 그려보기
여러가지 그래프를 그려 봅니다.
float y 값에 원하는 수식을 넣어 보세요
x 값은 p.x를 사용 합니다.
이해를 돕기 위해 y 값은 좌표계를 위로 갈수록 커지도록 변환
float2 p = uv * 2.0 - 1.0;
정규화 : xy : -1 ~ 1

자주 사용하는 그래프

  • 1. 직선 : float y = p.x;
  • 2. 사인 곡선 : float y = sin(p.x);
  • 3. Fract 곡선 (톱니파 모양) : float y = fract(sin(p.x));
    Fract 사용법
    fract(-1.3) = -2.0 + 0.7
    fract(1.3) = 1.0 + 0.3
  • 4. Step (계단 그래프) : float y = step(0.5, p.x);
    p *= 2.;
    float y = step(0.8, p.x);
    y -= step(1.2, p.x);
  • 4.1. SmoothStep float y = smoothstep(0.0, 1.0, p.x);
  • 5. 가우시안 곡선 : float y = exp(-p.x * p.x);
  • 5. 절대값 곡선 (대칭) : float y = abs(p.x);
  • 6. 제곱 곡선 : float y = p.x * p.x;
  • 7. 제곱근 : float y = sqrt(p.x);
  • 8. 코사인 : float y = cos(p.x);
  • 9. 탄젠트 : float y = tan(p.x);
  • 10. Traiangular Wave (지그재그) float y = abs(fract(p.x) * 2.0 - 1.0);
  • TIP
    좌표계 확대 방법 p *= 3.14;

핵심 개념

  • 정규화
    
float2 uv = fragCoord / iResolution.xy;
  • 원을 만드는 코드 1 
    
uniform float2 iResolution;
float circle(in vec2 _st, in float _radius) { 
    vec2 l = _st - vec2(0.5);
    return 1. - step(_radius, dot(l, l) * 4.0);
}
half4 main(float2 fragCoord) {
    float2 uv = fragCoord / iResolution.xy;    
    float color = circle(uv, 0.3);
    return half4(color, color, color, 1.0);
}

  • 패턴 만들기 1 
    
uniform float2 iResolution;
half4 main(float2 fragCoord) {
    float2 uv = (fragCoord / iResolution.xy);    
    uv *= 3.0;
    uv = fract(uv);
    float3 color = float3(uv, 0.0);
    return half4(color, 1.0);
}
  • 패턴 만들기 2 
    
vec2 tile(vec2 _st, float _zoom){
    _st *= _zoom;
    return fract(_st);
}

float box(vec2 _st, vec2 _size, float _smoothEdges){
    _size = vec2(0.5) - _size * 0.5;
    vec2 aa = vec2(_smoothEdges * 0.5);
    vec2 uv = smoothstep(_size, _size + aa, _st);
    uv *= smoothstep(_size, _size + aa, vec2(1.0) - _st);
    return uv.x * uv.y;
}     

uniform float2 iResolution;    
half4 main(float2 fragCoord) {
    vec2 st = fragCoord.xy / iResolution.xy;
    vec3 color = vec3(0.0);

    // Divide the space in 4
    st = tile(st,4.);

    // Draw a square
    color = vec3(box(st,vec2(0.7),0.01));
    // color = vec3(st,0.0);
    return vec4(color,1.0);
}
  • iResolution: 화면 해상도 (너비, 높이)
  • uv 좌표: 정규화된 좌표 (0.0~1.0 범위)
  • 그라디언트: 위치에 따라 색상이 점진적으로 변하는 효과
