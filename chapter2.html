랜덤 함수를 만들어지는 원리는 좋은 해시 함수를 만드는 것
12.9898, 78.233, 43758.5453123 과 같은 값은 마법의 상수.

랜덤 함수 (해시 함수) 생성 방법

1) 부동 소수점을 곱해준다. 43758.5453123
2) dot 함수를 사용하여 고차원을 스칼라 값으로 변환
4) fract, sin, cos 값을 적절하게 사용.

  • 노이즈 함수 1
    float y = mix(rand(p), rand(p + 1.0), p.x);
  • 노이즈 함수 2
    float y = mix(rand(p), rand(p + 1.0), smoothstep(0., 1., p.x));
  • 노이즈 함수 3
    float f = fract(p.x);
    float u = f * f * (3.0 - 2.0 * f );
    float y = mix(rand(p), rand(p + 1.0), u);
